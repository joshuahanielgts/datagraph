# 📊 DataGraph

**DataGraph** is a simple and flexible Python library for generating beautiful graphs from CSV files.

---

## 🚀 Features

- ✅ Supports major chart types:
  - Line chart
  - Bar chart
  - Scatter plot
  - Area chart
  - Histogram
  - Pie chart
- 🎯 Customizable output (title, size, palette)
- 🧪 Tested and reliable
- 🖥️ CLI support for easy terminal use
